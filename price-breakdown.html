Sketch

Sketch is a software program that allows you to design and prototype your web applications. My husband is a soft goods designer and uses Illustrator, so I've learned a little bit of that program, but am thinking about getting Sketch down the road because it has specific web design interfaces. I am working on just drawing out my own rough sketches currently, but for the table practice I decided to use this as an example. That way I can learn more about it!
